# Documentation Overview

This directory contains the top-level strategic documentation for AIOS: vision, roadmap, and general guidance for navigating the repository.

## Contents

| Document | Purpose |
|---|---|
| [`vision.md`](vision.md) | Long-term vision, design intent, assumptions, and non-goals |
| [`roadmap.md`](roadmap.md) | Phased delivery plan with outcomes, deliverables, and exit criteria |
| [`glossary.md`](glossary.md) | Core terminology definitions used across all AIOS artefacts |

## Relationship to other directories

`docs/` holds strategic intent. Decisions that implement that intent are recorded in [`adr/`](../adr/README.md). The architectural expression of the vision lives in [`architecture/`](../architecture/). Governance controls live in [`governance/`](../governance/). Knowledge management intent lives in [`knowledge/`](../knowledge/) and [`ontology/`](../ontology/).

## Traceability

All documents here carry a `DOC-` identifier as defined in the [Traceability Standard](../governance/traceability-standard.md).

- [`vision.md`](vision.md) — DOC-001
- [`roadmap.md`](roadmap.md) — DOC-002
